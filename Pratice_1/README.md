# Pratice 1

This folder keeps the first set of Python practice problems.

Some files are intentionally short and plain because they are practice notes.
`practice_set_q3_to_q9.py` is the reusable version that can be imported and
tested.

## Files

| File | What it practices |
|------|-------------------|
| `01.py` | Prime number checking |
| `fibo.py` | Recursive fibonacci with cache |
| `practice_set_q3_to_q9.py` | Q3-Q9 reusable solutions |
| `q3_two_sum.py` | Two sum |
| `q4_flatten.py` | Flatten nested list |
| `q5_anagram.py` | Group anagrams |
| `q6_bin_search.py` | Binary search |
| `q7_freq.py` | Most frequent element |

## Run

```powershell
python Pratice_1\practice_set_q3_to_q9.py
```
